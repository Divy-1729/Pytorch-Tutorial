# Pytorch-Tutorial

A tutorial for mastering Pytorch.

Pytorch is the most widely used library for building ML models. It also gives you much more flexibility than tensorflow. 

This tutorial (or collection of notes) is my attempt at distilling from this awesome [resource.](https://www.learnpytorch.io/00_pytorch_fundamentals/). I planned and wrote this in a span of about 14 days.

This repo is organized by chapter numbers in the original resource. Each chapter contains notes in a jupyter notebook and exercises generated by GPT. 

Future plan is to deploy this to the web.
